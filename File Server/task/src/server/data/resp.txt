
Чтобы создать новый файл на сервере, клиент должен отправить
запрос PUT NAME DATA, где NAMEуказано имя файла без пробелов или
 вкладок и DATAуказаны текстовые данные. Если файл с таким именем
 уже существует, сервер должен ответить кодом 403. Если файл создан успешно,
  сервер отвечает кодом 200. В этом случае клиент должен распечатать
   The response says that the file was created!. В противном случае он должен напечатать
    The response says that creating the file was forbidden!





Чтобы получить файл с сервера, клиент должен отправить запрос GET NAME,
 где NAME указано имя файла. Если файл с таким именем не существует, сервер должен ответить
  кодом 404. Если файл с таким именем существует, сервер отвечает кодом 200, одним пробелом
   и содержимым файла. В этом случае клиент должен распечатать
   The content of the file is: FILE_CONTENT, где FILE_CONTENT
   находится содержимое запрошенного файла. В противном случае он должен
   напечатать The response says that the file was not found!



Чтобы удалить файл с сервера, клиент должен отправить запрос DELETE NAME,
в котором NAMEуказано имя файла. Если файл с таким именем не существует, сервер
 должен ответить кодом 404. Если файл успешно удален, сервер ответит кодом 200.
 В этом случае клиент должен распечатать The response says that the file was successfully deleted!.
  В противном случае он должен напечатать The response says that the file was not found!
